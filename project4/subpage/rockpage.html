Rocks

These models were created as my introduction to Mudbox, done as a way to familiarize myself with the basic functionality of the program. I created their models entirely from scratch, utilizing the sculpting tools to form the shapes and apply finer details. After that, I applied some basic paint layers to color the rocks, applying special attention to detail to create the rocky surfaces.

Rock - High Poly

This rock model was created entirely from scratch in Mudbox, carved into a rocky shape using the program's sculpting tools. It was then colored using the various paint tools in Mudbox, layered with multiple different colors to give it a chalky sort of rock effect.

Rock - Low Poly
